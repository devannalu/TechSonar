# TechSonar — API

API backend do TechSonar, desenvolvida com **NestJS + TypeScript**.

## Status

🚧 Em fase de estruturação inicial.

## Stack

- NestJS
- TypeScript
- PostgreSQL
- Prisma (ORM)
- Pagar.me (pagamentos)
- Firebase Cloud Messaging (notificações)

## Estrutura planejada

A estrutura da API NestJS será definida durante o desenvolvimento, seguindo a arquitetura modular padrão do NestJS.

## Pré-requisitos

- Node.js (LTS)
- npm ou yarn
- PostgreSQL
- Prisma CLI

## Primeiros passos

O projeto NestJS ainda não foi inicializado. Será criado a partir daqui quando a fase de desenvolvimento da API iniciar.
