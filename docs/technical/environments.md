# Ambientes e Configurações

Documentação dos ambientes de execução do TechSonar.

## Status

🚧 Em fase de planejamento.

## Ambientes Planejados

| Ambiente      | Descrição                                |
| ------------- | ---------------------------------------- |
| Development   | Ambiente local de desenvolvimento        |
| Staging       | Ambiente de testes e homologação         |
| Production    | Ambiente de produção                     |

## Variáveis de Ambiente

As variáveis de ambiente serão documentadas conforme cada serviço for configurado.

Todas as variáveis sensíveis devem ser armazenadas em arquivos `.env` (não versionados) e nunca expostas no código fonte.
