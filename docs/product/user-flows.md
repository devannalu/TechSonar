# Fluxos do Usuário

## Fluxo do Participante

1. Criar conta
2. Fazer login
3. Ver eventos
4. Filtrar eventos
5. Abrir detalhes do evento
6. Fazer inscrição
7. Acessar ingresso
8. Fazer check-in
9. Enviar feedback
10. Receber certificado

## Fluxo do Organizador

1. Criar conta
2. Criar Perfil Organizador
3. Criar evento
4. Publicar evento
5. Acompanhar inscrições
6. Gerar QR Code do evento
7. Validar check-ins
8. Ver feedbacks
9. Emitir certificados
10. Acompanhar métricas
