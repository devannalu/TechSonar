# Decisões Oficiais

Decisões de produto e tecnologia aprovadas para o TechSonar.

## Lista de Decisões

1. O TechSonar será mobile-first.
2. O aplicativo será desenvolvido com Flutter + Dart.
3. A API será desenvolvida com NestJS + TypeScript.
4. O banco de dados será PostgreSQL.
5. O ORM será Prisma.
6. O gateway de pagamento será Pagar.me.
7. As notificações push serão feitas com Firebase Cloud Messaging.
8. Todo evento precisa estar vinculado a um Perfil Organizador.
9. Comunidade será um tipo de Perfil Organizador, não a entidade principal obrigatória.
10. O protótipo antigo será mantido isolado e não será usado como base de código.
