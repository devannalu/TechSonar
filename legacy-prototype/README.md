# Legacy Prototype

Este diretório contém o protótipo antigo do TechSonar.

## Finalidade

O protótipo é mantido aqui **apenas como referência**:

- Regras de negócio validadas
- Fluxos do usuário
- Estrutura de telas
- Experiência desejada
- Decisões de produto

## Regras

- **Não modifique** nenhum arquivo neste diretório.
- **Não refatore** o código do protótipo.
- **Não migre** código daqui para a aplicação real.
- **Não use** este código como base para `apps/mobile/` ou `services/api/`.

A nova aplicação será construída do zero, utilizando apenas as decisões de produto e regras de negócio como guia.
